# Security Policy

## Supported Versions

Only the latest release of `@tetrascience-npm/tetrascience-react-ui` receives security fixes.

| Version | Supported |
| --- | --- |
| Latest | Yes |
| Older releases | No |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report security issues by emailing [security@tetrascience.com](mailto:security@tetrascience.com). Include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected versions

You can expect an acknowledgement within 2 business days. If confirmed, a fix will be prioritised and a patched release coordinated before public disclosure.

For general questions, contact [cs@tetrascience.com](mailto:cs@tetrascience.com).
